# Database Schema\n\n## Collections\n\n### Users\n```javascript\n{\n  _id: ObjectId,\n  email: String (unique),\n  password: String (hashed),\n  name: String,\n  avatar: String,\n  bio: String,\n  role: String (teacher, admin, user),\n  school: String,\n  subject: String,\n  experience: Number,\n  followers: [ObjectId],\n  following: [ObjectId],\n  createdAt: Date,\n  updatedAt: Date\n}\n```\n\n### Stories\n```javascript\n{\n  _id: ObjectId,\n  title: String,\n  content: String,\n  excerpt: String,\n  author: ObjectId (ref: User),\n  category: String,\n  tags: [String],\n  likes: [ObjectId],\n  bookmarks: [ObjectId],\n  comments: [ObjectId],\n  viewCount: Number,\n  status: String (draft, published),\n  createdAt: Date,\n  updatedAt: Date\n}\n```\n\n### Comments\n```javascript\n{\n  _id: ObjectId,\n  story: ObjectId (ref: Story),\n  author: ObjectId (ref: User),\n  content: String,\n  likes: [ObjectId],\n  replies: [ObjectId],\n  parentComment: ObjectId,\n  createdAt: Date,\n  updatedAt: Date\n}\n```\n\n## Indexes\n- Stories: text search on title, content, tags\n- Stories: author + createdAt\n- Stories: category + status\n- Comments: story + createdAt\n
