# API Documentation\n\n## Base URL\n```\nhttp://localhost:5000/api\n```\n\n## Authentication\nUse JWT token in Authorization header:\n```\nAuthorization: Bearer <token>\n```\n\n## Endpoints\n\n### Auth\n- `POST /auth/register` - Register new user\n- `POST /auth/login` - Login\n- `GET /auth/profile` - Get profile (requires auth)\n- `POST /auth/logout` - Logout (requires auth)\n\n### Stories\n- `GET /stories` - Get all stories (paginated)\n- `GET /stories/:id` - Get story by ID\n- `POST /stories` - Create story (requires auth)\n- `PUT /stories/:id` - Update story (requires auth, owner only)\n- `DELETE /stories/:id` - Delete story (requires auth, owner only)\n- `GET /stories/user/:userId` - Get user's stories\n- `POST /stories/:id/like` - Like story (requires auth)\n- `POST /stories/:id/bookmark` - Bookmark story (requires auth)\n\n### Comments\n- `POST /comments/story/:storyId` - Add comment (requires auth)\n- `GET /comments/story/:storyId` - Get comments\n- `PUT /comments/:id` - Update comment (requires auth, author only)\n- `DELETE /comments/:id` - Delete comment (requires auth, author only)\n- `POST /comments/:id/like` - Like comment (requires auth)\n\n## Response Format\nAll responses are JSON objects.\n\n### Success Response\n```json\n{\n  \"data\": { /* response data */ },\n  \"status\": 200\n}\n```\n\n### Error Response\n```json\n{\n  \"message\": \"Error description\",\n  \"status\": 400\n}\n```\n
