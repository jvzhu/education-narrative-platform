# Deployment Guide\n\n## Using Docker Compose (Local Development)\n\n```bash\ngit clone https://github.com/jvzhu/education-narrative-platform.git\ncd education-narrative-platform\ndocker-compose up\n```\n\nAccess:\n- Frontend: http://localhost:3000\n- Backend: http://localhost:5000\n- MongoDB: mongodb://localhost:27017\n\n## Production Deployment\n\n### Frontend (Vercel)\n1. Push code to GitHub\n2. Connect repo to Vercel\n3. Set environment variables:\n   - `VITE_API_URL=https://your-api-url/api`\n4. Deploy\n\n### Backend (Railway/Heroku)\n1. Create app on Railway/Heroku\n2. Set environment variables:\n   - `MONGODB_URI=your-mongodb-atlas-uri`\n   - `JWT_SECRET=your-secure-secret`\n3. Deploy\n\n### Database (MongoDB Atlas)\n1. Create cluster on MongoDB Atlas\n2. Create database user\n3. Get connection string\n4. Update backend `MONGODB_URI`\n\n## Environment Variables\n\n**Frontend (.env)**\n```\nVITE_API_URL=http://localhost:5000/api\n```\n\n**Backend (.env)**\n```\nNODE_ENV=production\nPORT=5000\nMONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/education-narrative\nJWT_SECRET=your-secret-key\nCORS_ORIGIN=https://your-frontend-url\n```\n\n## CI/CD with GitHub Actions\nThe `.github/workflows/ci.yml` file automatically:\n1. Tests backend code\n2. Builds frontend\n3. Builds Docker images\n4. Prepares for deployment on push to main\n\n## Health Check\n```bash\ncurl http://localhost:5000/health\n```\n\nExpected response:\n```json\n{\n  \"status\": \"healthy\",\n  \"timestamp\": \"2024-01-01T00:00:00Z\"\n}\n```\n
