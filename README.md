# 🎓 教育叙事平台 (Education Narrative Platform)\n\n> 在文字的世界里遇见教育的美好，书写幸福的篇章\n\n![License](https://img.shields.io/badge/license-MIT-blue.svg)\n![Node](https://img.shields.io/badge/node-20.x-green.svg)\n![MongoDB](https://img.shields.io/badge/mongodb-6.0-green.svg)\n![React](https://img.shields.io/badge/react-18.x-blue.svg)\n\n## ✨ 功能特性\n\n### 📝 故事发布\n- ✅ 发布和编辑教育故事\n- ✅ 支持多种分类（教学、反思、阅读、学生成长等）\n- ✅ 标签系统便于分类和搜索\n- ✅ 草稿和发布功能\n\n### 💬 社区互动\n- ✅ 点赞和收藏功能\n- ✅ 评论和讨论\n- ✅ 关注教师功能\n- ✅ 浏览统计\n\n### 👥 用户管理\n- ✅ 用户注册和登录\n- ✅ 个人资料管理\n- ✅ 教师认证\n- ✅ 用户权限管理\n\n### 🔐 安全特性\n- ✅ JWT 身份验证\n- ✅ 密码加密存储\n- ✅ CORS 配置\n- ✅ 输入验证\n\n## 🏗️ 技术栈\n\n### Frontend\n- **框架**: React 18 + Vite\n- **样式**: Tailwind CSS\n- **状态管理**: Zustand\n- **HTTP 客户端**: Axios\n- **数据查询**: React Query\n- **路由**: React Router v6\n- **UI 组件**: React Icons\n- **日期处理**: date-fns\n\n### Backend\n- **运行时**: Node.js 20\n- **框架**: Express.js\n- **数据库**: MongoDB\n- **认证**: JWT (jsonwebtoken)\n- **密码**: bcryptjs\n- **CORS**: CORS 中间件\n- **环境配置**: dotenv\n\n### DevOps\n- **容器化**: Docker & Docker Compose\n- **CI/CD**: GitHub Actions\n- **反向代理**: Nginx\n- **Web 服务器**: Node.js + Express\n\n## 🚀 快速开始\n\n### 前置需求\n- Node.js 20+\n- Docker & Docker Compose (可选)\n- MongoDB (本地或 MongoDB Atlas)\n- Git\n\n### 本地开发\n\n#### 使用 Docker Compose (推荐)\n```bash\n# 克隆仓库\ngit clone https://github.com/jvzhu/education-narrative-platform.git\ncd education-narrative-platform\n\n# 启动所有服务\ndocker-compose up\n\n# 访问\n# 前端: http://localhost:3000\n# 后端: http://localhost:5000\n# 数据库: mongodb://localhost:27017\n```\n\n#### 手动启动\n\n**1. 后端设置**\n```bash\ncd backend\nnpm install\n\n# 创建 .env 文件\ncat > .env << EOF\nNODE_ENV=development\nPORT=5000\nMONGODB_URI=mongodb://localhost:27017/education-narrative\nJWT_SECRET=your-secret-key-change-in-production\nCORS_ORIGIN=http://localhost:3000\nEOF\n\n# 启动服务器\nnpm start\n```\n\n**2. 前端设置**\n```bash\ncd frontend\nnpm install\n\n# 创建 .env 文件\ncat > .env << EOF\nVITE_API_URL=http://localhost:5000/api\nEOF\n\n# 启动开发服务器\nnpm run dev\n```\n\n**3. 启动 MongoDB**\n```bash\n# 使用 Docker\ndocker run -d -p 27017:27017 --name education-db mongo:6.0\n\n# 或使用本地 MongoDB\nmongod\n```\n\n## 📁 项目结构\n\n```\neducation-narrative-platform/\n├── frontend/   - 用户登录\nGET    /api/auth/profile        - 获取用户资料\nPOST   /api/auth/logout         - 用户登出\n```\n\n### 故事 (Stories)\n```\nGET    /api/stories             - 获取所有故事（分页）\nGET    /api/stories/:id         - 获取单个故事\nPOST   /api/stories             - 发布新故事\nPUT    /api/stories/:id         - 更新故事\nDELETE /api/stories/:id         - 删除故事\nGET    /api/stories/user/:id    - 获取用户的故事\nPOST   /api/stories/:id/like    - 点赞故事\nPOST   /api/stories/:id/bookmark - 收藏故事\n```\n\n### 评论 (Comments)\n```\nPOST   /api/comments/:storyId   - 添加评论\nGET    /api/comments/:storyId   - 获取评论\nPUT    /api/comments/:id        - 更新评论\nDELETE /api/comments/:id        - 删除评论\nPOST   /api/comments/:id/like   - 点赞评论\n```\n\n更详细信息见 [API 文档](docs/API.md)\n\n## 🧪 测试\n\n```bash\n# 后端测试\ncd backend\nnpm
